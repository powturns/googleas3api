public function search(searchString:String, startValue:int = 0, safeMode:String, size:String = "", colorization:String = "", imageColor:String = "", imageType:String = "", filetype:String = "", restrictToCreativeCommons:Boolean = false, lang:String = "", restrictToDomain:String = ""):void

Start search operation, when operation is done, will dispatch an GoogleSearchEvent.IMAGE_SEARCH_RESULT event with an array of GoogleImage-objects. Will deliver 8 results max, you can get more using the startValue param. (paging)

Parameters
searchString:String — search string, what do you wanna find?
 
startValue:int (default = 0) — sets a start value for paging (fe. 9; delivers search results from result 9 up to 16)
 
safeMode:String — This optional argument supplies the search safety level, you can find all values in GoogleImageSafeMode (package data.type)
 
size:String (default = "") — Restrict results to images of a certain size, you can find all values in GoogleImageSize (package data.type)
 
colorization:String (default = "") — Restrict results to images of a certain colorization, you can find all values in GoogleImageColorization (package data.type)
 
imageColor:String (default = "") — NEW This optional argument tells the image search system to filter the search to images of the specified color, you can find all allowed values in GoogleImageColor (package data.type)
 
imageType:String (default = "") — Restrict results to images of a certain type, you can find all values in GoogleImageType (package data.type)
 
filetype:String (default = "") — Restrict results to images with a certain extension, you can find all values in GoogleImageFiletype (package data.type)
 
restrictToCreativeCommons:Boolean (default = false) — Restrict results to CreativeCommons licensed images
 
lang:String (default = "") — set main language using language code
 
restrictToDomain:String (default = "") — NEW This optional argument tells the image search system to restrict the search to images within the specified domain
Note: This restriction may restrict results to images found on pages AT the given URL and/or images WITH the given URL.
